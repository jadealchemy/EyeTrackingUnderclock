Error Code Handling

  • RUNTIME_NOT_FOUND
    • Please install SR_Runtime from below link: https://dl.vive.com/SRSDK/runtime/VIVE_SRanipalInstaller.msi\
  • NOT_INITIAL:
    • Please call ViveSR::anipal::Initial() before use any SRanipal API.
  • INVALID_INPUT
    • The given parameter is not support.
  • INITIAL_FAILED
    • The initial function failed. Please check your hardware are set properly or firmware is in latest version.
  • UNINSTALL_STEAM
    • Please install SteamVR first.
  • INVALID_CHANGE
    • The given parameter is not support.
  • TIMEOUT
    • Runtime connection timeout. Please reboot SR_Runtime or Client side application.
  • DEVICE_NOT_FOUND
    • Please check your hardware are set properly.
  • NOT_AUTHORIZED
    • User dont have enough permission to use this API, please contact application supplier.
